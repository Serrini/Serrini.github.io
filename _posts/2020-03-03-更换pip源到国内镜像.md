# 更换pip源到国内镜像

* 修改 pip.conf 文件

路径`$HOME/.config/pip/pip.conf`

没有该文件自己创建一个

常用命令

ls -a 显示所有文件（包括隐藏文件）

mkdir 创建文件夹

touch pip.conf 创建文件

vi pip.conf

* pip.conf修改成

```conf
[global]
timeout = 6000
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn
```

* 国内的其他镜像源
